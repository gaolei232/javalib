设计要点

  • 温暖专业风 — 从原版冷灰蓝 neumorphism 转为暖色系
  • 侧栏导航 — 保持原有导航结构，暖色背景 + 选中态琥珀色高亮
  • Hero 卡片 — 蜂蜜黄渐变背景替代原版灰白，更温暖
  • 统计卡 — 四色图标区分不同指标类型
  • 数据表格 — 极简表格设计，暖色状态标签
  • 所有功能模块全保留 — 数据总览、预约管理、用户管理、座位管理、学生统计、系统设置
  • API 照搬 — seatApi / studentStatsApi / userApi 全部接口调用不变
