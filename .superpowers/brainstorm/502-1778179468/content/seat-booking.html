设计要点

  • 三段式布局 — 顶部导航、筛选区、地图+详情+时间线
  • 筛选面板合一 — 楼栋/楼层/日期 + 靠窗/插座/安静区整合到一行
  • 座位色阶 — 暖沙色(可约) → 蜂蜜黄(部分占用) → 琥珀棕(我已约) → 灰色(已满)
  • 详情面板 — 右侧固定详情区，含座位信息、时段列表、操作按钮
  • 时间线简化 — 14段紧凑时间轴，颜色对应状态
  • API 照搬 — seatApi.getAllSeats / bookSeat / cancelBookingById / getUserBookedSeats / getSeatBookingsByDate 全部照搬
