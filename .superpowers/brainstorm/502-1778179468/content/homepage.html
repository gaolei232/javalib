设计要点

  • 导航简化 — 移除冗余标签，LOGO + 登录/注册两个 CTA 按钮
  • Hero 聚焦 — 大标题 + 温暖文案，营造图书馆氛围
  • 数据轻量 — 三个关键数字替代原来的指标卡片矩阵
  • 功能卡片 — 三张柔和卡片展示核心能力，暖色图标区域
  • 删除右侧系统亮点卡片 — 信息整合进 Hero 和底部 Features
  • API 接口照搬 — 路由链接、跳转逻辑保持不变
